## Limitations

Compatible with the Operating Systems:

    Debian: 5.0, 6.0
    RedHat: 5.0, 6.0, 7.0
    SLES
    Ubuntu: 14.04, 12.04, 10.04

This module does not...

* open up tcp port 8000 on the local firewall
  
* add a SELinux policy to permit binding to port 8000
